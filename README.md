# MEL Flight Explorer

Interactive flight planning and route visualisation tool for a family trip from the UK to Melbourne, Australia over Christmas 2025.

## What is this?

A single-page web application that helps a family of four (2 adults + 2 children aged 4 and 7) discover, compare, and book the best flight routes from UK airports to Melbourne Tullamarine (MEL) for Christmas 2025 travel.

## Features

- **Interactive World Map** with animated arc lines colour-coded by price band
- **Multiple UK Departure Airports** - MAN, BHX, LHR with instant switching
- **17 Stopover Hubs** - Middle East, Asian, European, and unconventional hubs
- **Route Cards** with airlines, duration, stops, quality, and family pricing
- **Deep Booking Links** to Google Flights and Skyscanner pre-filled with passengers and dates
- **Unconventional Routes** highlighted with estimated savings
- **Smart Filtering** by airport, date, budget, and stops
- **Two Sort Modes** - Best Price and Best Route
- **Map-Card Linking** - interactive highlighting between map arcs and route cards
- **Fully Responsive** - desktop, tablet, and mobile

## Trip Details

- 2 adults + 2 children (ages 4 and 7)
- Outbound: 25-28 December 2025 | Return: by 14 January 2026
- Budget: approx 1000/person (4000-4600 total) | Economy class
- Destination: Melbourne Tullamarine (MEL)

## Tech Stack

Single self-contained HTML file. Leaflet.js (CDN), CartoDB Dark Matter tiles, Google Fonts. All data hardcoded as JSON. Zero backend, zero API keys.

## Usage

Open index.html in any modern browser. No server required.

## Disclaimer

Prices shown are indicative estimates. Click Search This Route on any card to see live pricing.
